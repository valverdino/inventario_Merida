# Inventario de Arbolado de Mérida 🌳

Proyecto de inventario y visualización del arbolado en zonas verdes y viario de Mérida, generado con QGIS y publicado con qgis2web.

## 🌐 Páginas publicadas

| Recurso | Enlace |
|---|---|
| 🗺️ Mapa web interactivo | [ver mapa](https://valverdino.github.io/inventario_Merida/) |
| 📊 Dashboard del inventario | [ver dashboard](https://valverdino.github.io/inventario_Merida/dashboard_arbolado_V2.html) |

## 📦 Contenido del repositorio

- `index.html` — Mapa interactivo Leaflet generado con qgis2web
- `dashboard_arbolado_V2.html` — Panel de estadísticas del inventario
- `data/` — Capas de datos en formato GeoJSON (frondosas, coníferas, palmeras, callejero, barrios)
- `css/` — Estilos del mapa
- `js/` — Librerías JavaScript (Leaflet y plugins)
- `markers/` — Iconos SVG por tipo de árbol
- `legend/` — Imágenes de leyenda
- `webfonts/` — Fuentes tipográficas

## 🛠️ Tecnologías

- [QGIS](https://qgis.org) + [qgis2web](https://github.com/tomchadwin/qgis2web)
- [Leaflet.js](https://leafletjs.com)
